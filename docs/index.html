BioMiner Indexd

A hash-based data indexing and tracking service providing globally unique identifiers for omics data files. Similar to Indexd, but with enhanced features for biomedical research.

Explore Features View Datasets

Key Features

Powerful tools for managing and tracking omics data

Global Unique Identifiers

Generate and manage globally unique identifiers for each data file with comprehensive metadata tracking.

File Location Tracking

Track file locations across multiple repositories (OSS, S3, GSA, NODE, SRA, ENA) with automatic synchronization.

Multi-Version Management

Manage different versions of files with Base UUID indexing for comprehensive version control.
